# Руководство по кастомизации Horae

## Быстрый старт

Все визуальные стили Horae управляются **CSS-переменными**. Достаточно переопределить эти переменные, чтобы изменить внешний вид всего плагина.

### Способ 1: Изменение CSS-переменных (рекомендуется)

Введите в настройках плагина → Настройки внешнего вида → Пользовательский CSS:

```css
#horae_drawer,
.horae-message-panel,
.horae-modal,
.horae-context-menu,
.horae-progress-overlay {
    --horae-primary: #ec4899;      /* Основной цвет → розовый */
    --horae-primary-light: #f472b6;
    --horae-bg: #1a1020;           /* Фон → тёмно-фиолетовый */
    --horae-bg-secondary: #2d1f3c;
}
```

### Способ 2: Импорт файла кастомизации

1. Получите `.json`-файл темы, которым поделились другие
2. Настройки плагина → Настройки внешнего вида → нажмите кнопку импорта (📥)
3. Выберите импортированную тему из выпадающего списка

### Способ 3: Экспорт и публикация

1. Настроив понравившийся стиль, нажмите кнопку экспорта (📤)
2. Скачается файл `horae-theme.json`
3. Поделитесь им с другими пользователями

## Формат файла кастомизации

Структура экспортируемого `.json`-файла:

```json
{
    "name": "Моя тема",
    "author": "Ваше имя",
    "version": "1.0",
    "variables": {
        "--horae-primary": "#ec4899",
        "--horae-primary-light": "#f472b6",
        "--horae-primary-dark": "#be185d",
        "--horae-accent": "#f59e0b",
        "--horae-bg": "#1a1020",
        "--horae-bg-secondary": "#2d1f3c",
        "--horae-bg-hover": "#3c2f50",
        "--horae-border": "rgba(255, 255, 255, 0.08)",
        "--horae-text": "#e5e5e5",
        "--horae-text-muted": "#a0a0a0"
    },
    "css": "/* Необязательно: дополнительный CSS */\n.horae-timeline-item { border-radius: 12px; }"
}
```

**Описание полей:**
- `name`: Название темы (отображается в селекторе тем)
- `author`: Имя автора (необязательно)
- `version`: Номер версии (необязательно)
- `variables`: Пары ключ-значение CSS-переменных, переопределяющие значения по умолчанию
- `css`: Дополнительный CSS-код (необязательно) для стилей, не реализуемых через переменные

## Частые вопросы и советы

### Нижняя панель перекрывается другими элементами (невозможно взаимодействовать)

Некоторые темы или пресеты таверны имеют высокий z-index, из-за чего нижняя панель Horae оказывается под ними. Добавьте в пользовательский CSS:

```css
.horae-message-panel {
    margin-bottom: 10px;
    z-index: 9999;
    position: relative;
}
```

### Замена иконки верхнего ящика

Замените иконку Horae в верхней навигации на собственное изображение:

```css
#horae_drawer .drawer-icon::before {
    background-image: url('URL вашего изображения') !important;
}
```

## Важные замечания

1. **Область видимости переменных**: CSS-переменные определяются в контейнерах верхнего уровня — `#horae_drawer`, `.horae-modal` и т. д. Не определяйте их на `body` или `:root` — не сработает.

2. **Защита `!important`**: некоторые стили кнопок имеют `!important` для защиты от глобальных тем таверны. Чтобы переопределить такие стили, в пользовательском CSS тоже нужен `!important`.

3. **Светлый/тёмный режим**: после выбора пользовательской темы она переопределяет переменные тёмного/светлого режима по умолчанию. Если ваша тема светлая, не забудьте сделать `--horae-text` тёмным.

4. **Не влияет на таверну**: все стили Horae ограничены контейнерами плагина и не затрагивают основной интерфейс таверны.
